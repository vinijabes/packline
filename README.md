# Packline ![Github Actions status](https://github.com/vinijabes/packline/workflows/Rust/badge.svg)

Packline will be a fast, reliable and scalable message broker, built in rust.

# Running
## Build
```sh
cargo build
```

## Run
```sh
cargo run
```
